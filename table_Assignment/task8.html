Description of Table Assignment

In this assignment, we have used HTML hyperlinks to connect multiple web pages. Hyperlinks are created using the <a> (anchor) tag, which allows users to navigate from one page to another easily.

Each task is linked to a separate HTML file such as task1.html, task2.html, etc. By clicking on the "Open" link, the user can access the respective task page. The href attribute is used within the anchor tag to specify the destination.
