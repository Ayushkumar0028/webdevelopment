Description Link Assignment

In this assignment, we have used HTML links to connect different web pages. Links are created using the <a> (anchor) tag, which allows users to navigate from one page to another.

Each task is linked to a separate HTML file (task1.html, task2.html, etc.), making it easy to open and access different pages by clicking on the links. The href attribute is used inside the anchor tag to define the destination of the link.
