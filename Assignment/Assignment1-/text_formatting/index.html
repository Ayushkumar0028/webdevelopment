Description of Text Formatting

In this assignment, we have used different HTML text formatting tags to change the appearance of text on a web page.

Text formatting tags such as <b>, <i>, <u>, <strong>, and <em> are used to make text bold, italic, underlined, and more meaningful.

The main objective of this assignment is to understand how to format text using HTML and improve the readability and presentation of content.
